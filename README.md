# Arduino-plant

This code for Arduino ESP8266 which have a WiFi module, DHT11 sensor, light sensor, hygrometer and 0.96 OLED display 128Ñ…64. The device is able to connect to WiFi, get data from sensors, display data on OLED, send data to server and get data from server. 

## Fuetures

- Connect to WiFi
- Get data from sensors
- Display data on OLED
- Send data to server
- Get data from server

## Conetions

- DHT11 -> D5
- Hygrometer -> A0
- Light sensor -> D2
- OLED -> SDA -> D2, SCL -> D1
